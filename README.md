📚 EduPlatform – Online Learning Platform Backend

EduPlatform is a scalable backend system for an online learning platform that manages courses, enrollments, content delivery, and progress tracking.

The platform supports role-based access for Students and Instructors and provides APIs for authentication, course management, analytics, and user progress tracking.

🚀 Features
JWT Authentication & Authorization
Role-Based Access Control
Course & Lesson Management
Student Enrollment System
Progress Tracking
Video/Content Delivery Support
Analytics & Reporting APIs

🛠️ Tech Stack
Node.js / Flask
Express.js
MongoDB / MySQL
JWT Authentication
REST APIs
📂 Main Modules

Authentication Module
Course Management Module
Enrollment Module
Progress Tracking Module
Analytics Module
🔗 API Endpoints

POST /register
POST /login

GET /courses
POST /courses

POST /enroll

POST /progress/update
GET /progress/:userId
⚙️ Installation
git clone <repository-url>

npm install

npm start
